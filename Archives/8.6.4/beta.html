Soar 8.6.4 Beta

Note: this is not a full release of Soar and is not intended for wide distribution.

Windows

Right click and save: SoarSuite-8.6.4-beta-win32.zip

Unzip to a folder of your choice. Note: the demos have some long filenames and this may cause an error during extraction, it is safe to ignore this error. To avoid this error, extract to C:\ or C:\Soar

Use the included batch files (run-Debugger, run-VisualSoar, run-TankSoar, run-Eaters) to launch Soar.

Sun Java version 5 (1.5) or newer is required if you don't have it already: java.sun.com

To see what version of Java you have installed, run Start Menu / Run / cmd and type: 

java -version

If you have a newer version installed than what is being reported by that command, try uninstalling previous versions.

Mac OS X Intel Platform

Save and extract to your desktop: SoarSuite-8.6.4-beta-osx.tgz

Start a terminal and change to the folder you just extracted, usually: 

cd Desktop/SoarSuite-8.6.4-beta-osx

Launch Soar by running the files in that folder inside the Terminal window, note the "./" prefix:

./run-Debugger.sh
./run-TankSoar.sh
./run-Eaters.sh
./run-VisualSoar.sh

Linux

Save and extract to a folder of your choice: SoarSuite-8.6.4-beta-linux.tgz

Change to the folder you just extracted and run the scripts to launch Soar applications.

./run-Debugger.sh
./run-TankSoar.sh
./run-Eaters.sh
./run-VisualSoar.sh

Note: Sun Java 1.5 or better must be installed. These applications are known to not work with other, non-Sun runtimes. Try them at your own risk.

To upgrade Java, please consult your distribution documentation as the procedure varies widely depending on what flavor of Linux you are using.
